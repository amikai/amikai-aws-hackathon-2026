# Primary User Journey：市場越激動，介面越安定

> 文件狀態：唯一 Demo 主流程  
> Persona：小安，大盤 ETF 定期投資新手  
> Demo 日期：2025/04/09  
> 標的：0050 元大台灣 50  
> 原定計畫：每月 10 日投入 5,000 元  
> 核心情境：投入日前夕，因連續劇烈下跌而想暫停

## 1. 這次只驗證一件事

> 當市場劇烈波動、使用者想偏離原定計畫時，一隻真的會動的 3D 電子寵物，能否用安定的環境、動作與互動，陪使用者慢下來並做出有意識的選擇？

情緒弧線：

```text
焦慮 → 被接住 → 產生好奇 → 理解一件事 → 自主選擇 → 安心離開
```

核心體驗原則：

> 市場越激動，介面越安定。

## 2. 真實資料情境

2025/04/09 的 0050：

- 收盤價：146.2。
- 當日漲跌幅：-4.6%。
- 週報酬率：-16.931%。
- 月報酬率：-21.546%。
- 三大法人買賣超合計：-42,058.918 張。
- 4/7、4/8、4/9 三日法人皆為賣超。
- 論壇發文：196 則，發文人數 135 人。
- 看多 65、看空 10、中性 121。
- 回文 1,069 則，回文人數 654 人。
- 4/1–4/6 每日發文為 49–86 則，4/9 聲量明顯提高。

重要解讀：

> 市場價格劇烈下跌、法人持續賣超、論壇明顯變吵；但論壇內容以中性為主，不能說成「大家一致看空」。

## 3. 3D 小屋與電子寵物

### 3.1 畫面中心

- 固定鏡位的 3D 小屋，避免使用者迷失在自由鏡頭。
- 電子寵物位於視覺中心，占主要注意力。
- 資料不先做成 Dashboard，而是成為環境變化。
- 詳細數字只有在使用者主動探索後才出現。

### 3.2 小屋資料映射

| 小屋物件 | 資料意義 |
|---|---|
| 窗外風雨 | 價格波動與跌幅 |
| 室內暖燈 | 安定感，不隨損益轉紅綠 |
| 收音機音量與波紋 | 論壇發文與參與人數 |
| 收音機三個頻道 | 看多、看空、中性 |
| 門口大腳印 | 法人買賣方向與連續性 |
| 牆上路線圖 | 小安每月 10 日投入的原定計畫 |
| 小腳印 | 小安過去的實際投入紀錄 |
| 書桌日記 | 市場事實、感受、選擇與共同記憶 |

### 3.3 寵物動畫最低需求

1. `idle_breathe`：平穩呼吸與眨眼。
2. `approach_user`：發現使用者後靠近鏡頭。
3. `close_window`：高波動時關小窗戶，不表現恐慌。
4. `sit_together`：使用者選擇先坐坐時安靜陪坐。
5. `inspect_footprints`：拿放大鏡觀察法人腳印。
6. `offer_warm_drink`：探索前或焦慮時遞出熱飲。
7. `write_diary`：記錄使用者最後的選擇。
8. `put_away_diary`：結束時收起日記並回到平穩呼吸。

若製作時間不足，優先保留呼吸、靠近、關窗、陪坐與寫日記。

## 4. 情緒色彩系統

色彩不是直接重現漲跌，而是幫助使用者調節情緒。

### 4.1 基礎色調

- 奶油白：房間與主要背景。
- 溫暖米色：書桌、日記與紙張。
- 鼠尾草綠：穩定與長期成長。
- 霧藍色：窗外市場與雨天。
- 暖琥珀色：燈光、陪伴與注意力焦點。
- 柔和珊瑚色：非緊急提醒。

### 4.2 高波動狀態

市場高波動時：

- 窗外轉為較深的霧藍與灰藍。
- 雨勢與窗簾擺動增加，但設定強度上限。
- 室內暖琥珀燈光稍微增強。
- UI 資訊密度降低，一次只顯示一個選擇。
- 寵物動作變慢、靠近使用者，而不是驚慌跳動。

避免：

- 全畫面紅色。
- 閃爍警告。
- 急促震動。
- 寵物哭泣、生病或失控。
- 將下跌等同失敗。

市場方向只影響戶外的細微冷暖；介面的安定程度主要由「市場激動程度」決定，而不是由漲或跌決定。

## 5. 8 格 Storyboard

### 第 1 格：抵達小屋

**畫面**：窗外有雨與風，室內維持奶油白與暖琥珀燈。寵物平穩呼吸，看見小安後慢慢靠近。  
**動畫**：`idle_breathe → approach_user`  
**台詞**：

> 「你來了。今天外面的風有點大，我幫你把重要的事情留在桌上了。」

**使用者選擇**：

- 陪我看一下。
- 今天只想坐坐。

**目的**：先讓使用者被接住，不先顯示跌幅。

### 第 2 格：先安定，不急著分析

**Demo 路徑**：小安選擇「陪我看一下」。  
**畫面**：寵物先關小一半窗戶，再遞出熱飲。  
**動畫**：`close_window → offer_warm_drink`  
**台詞**：

> 「我們不用一次看完。先從一件事情開始就好。」

**替代路徑**：若選擇「今天只想坐坐」，寵物執行 `sit_together`，不再推送事件，可直接進入第 8 格的安心結束。

### 第 3 格：想起原本的路線

**畫面**：牆上地圖亮起，顯示每月 10 日的一個小節點；明天的節點尚未完成。  
**資料**：每月 10 日投入 5,000 元、十年以上長期累積。  
**台詞**：

> 「我記得我們原本約好明天一起往前走一步。今天你是不是有點想改變路線？」

**選項**：

- 我想暫停這次投入。
- 我只是有點擔心。
- 我的生活需求改變了。
- 我還說不上來。

**Demo 選擇**：我只是有點擔心。

### 第 4 格：聽見市場的聲音

**畫面**：收音機波紋變多，但聲音維持柔和。寵物坐在旁邊，讓使用者旋轉頻道。  
**互動**：切換「有多少人在說話」與「他們的立場」。  
**逐步揭露**：

- 196 則發文、135 位發文者。
- 看多 65、看空 10、中性 121。

**台詞**：

> 「外面確實比平常吵很多。不過仔細聽，大部分聲音其實沒有明確看多或看空。」

**學習點**：聲量增加不等於形成看空共識。

### 第 5 格：觀察門口腳印

**畫面**：門外有連續三日往外的大腳印。寵物拿起放大鏡蹲下觀察。  
**動畫**：`inspect_footprints`  
**逐步揭露**：

- 4/9 三大法人合計賣超 42,058.918 張。
- 4/7–4/9 都是賣超。

**台詞**：

> 「這些腳印是真的，而且連續往外走了幾天。但留下腳印的人，可能和我們有不同的目的與時間。」

**學習點**：法人是線索，不是必須跟隨的答案。

### 第 6 格：把時間尺度放在一起

**畫面**：桌面只留下兩張卡：左邊是「最近一週」，右邊是「原本十年計畫」。寵物坐在兩張卡中間。  
**資料**：0050 週報酬 -16.931%；小安原定計畫十年以上。  
**互動**：將收音機聲音與腳印線索拖到「短期市場」卡；將投入目的拖到「我的計畫」卡。  
**台詞**：

> 「短期的變化很大，讓人擔心很正常。它可能是重新檢視計畫的理由，但不必自動替你做決定。」

**目的**：不否定市場，也不要求維持投入。

### 第 7 格：把選擇交還給小安

**畫面**：牆上路線圖重新亮起，沒有綠色正解或紅色錯誤。寵物安靜坐在一旁。  
**選項**：

- 我想維持原本計畫。
- 我想有理由地調整計畫。
- 我想先記下感受，明天再確認。
- 我現在還沒有答案。

**Demo 建議選擇**：我想先記下感受，明天再確認。  
**台詞**：

> 「可以。暫時不決定，也是一個有意識的選擇。」

### 第 8 格：共同日記與安心離開

**畫面**：寵物在書桌寫下日記，然後把日記輕輕收好；窗外仍下雨，但室內維持安定。  
**動畫**：`write_diary → put_away_diary → idle_breathe`  
**日記內容**：

```text
2025/04/09
明天是原定投入日。
今天 0050 波動很大，論壇也明顯變吵；但多數發文是中性。
法人連續三日賣超。
小安感到擔心，選擇先記錄感受，明天再確認計畫。
```

**結束台詞**：

> 「今天我們只看了一件事，已經很夠了。你的計畫還在這裡，我也會幫你記得。」

**唯一主按鈕**：今天先到這裡。

## 6. 寵物互動狀態

```text
IDLE
  → ARRIVAL
  → GROUNDING
  → PLAN_RECALL
  → RADIO_EXPLORE
  → FOOTPRINT_EXPLORE
  → REFLECTION
  → DIARY
  → REST
```

必要分支：

```text
ARRIVAL
  → SIT_TOGETHER
  → REST
```

任何狀態都必須允許使用者選擇「今天先到這裡」，直接進入不帶責備的 REST。

## 7. MVP 畫面需求

只製作：

- 固定鏡位 3D 小屋。
- 一隻可動的 3D 電子寵物。
- 窗戶、暖燈、收音機、門口、路線圖、書桌與日記。
- 八組以內的角色動畫。
- 一條固定 Demo 資料路徑。
- 四個反思選項。
- 一個安心結尾。

暫不製作：

- 自由 3D 探索。
- 多隻寵物與換裝。
- 多股票切換。
- 完整養成商城。
- 開放式 AI 聊天。
- 即時行情。
- 多個市場事件同時出現。

## 8. 驗證問題

測試後詢問：

1. 你有感覺寵物真的在陪你，而不只是裝飾嗎？
2. 高波動畫面讓你更焦慮，還是比較能慢下來？
3. 你是否理解「聲量增加不等於人人看空」？
4. 你是否覺得產品在暗示你一定要繼續投入？
5. 哪一個寵物動作最有溫度？
6. 哪一段最像在上課或說教？
7. 結束時是否有完整、安心的感覺？

成功標準：

> 使用者感覺被陪伴、沒有被評價，理解一個市場現象，並知道最後的選擇仍屬於自己。
